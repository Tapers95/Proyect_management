# Proyect_management
Repositorio centralizado de inteligencia de proyectos. Este espacio integra cuadernos de NotebookLM y documentación clave para la gestión sistémica de proyectos, programas y portafolios. Nuestro objetivo es optimizar la toma de decisiones mediante el análisis profundo de datos y la planificación estructurada de cronogramas y recursos.
